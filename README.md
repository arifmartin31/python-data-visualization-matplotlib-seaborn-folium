#  Data Visualization Using Matplotlib, Seaborn & Folium

This project was completed as part of the IBM Data Science Professional Certificate.  
It demonstrates exploratory data analysis and visualization techniques using multiple Python libraries.

---

##  Project Overview

The objective of this notebook was to explore datasets and create meaningful visualizations using different visualization tools in Python. The project compares traditional plotting, statistical visualization, and geospatial mapping approaches.
